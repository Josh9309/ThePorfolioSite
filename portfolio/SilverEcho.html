Silver Echo

Summary

Silver Echo is a game that was made during the Global Game Jam 2017 at the RIT Jam Site. Silver Echo is a stealth game where you are in a mansion and you must escape without getting caught by security patrols. The catch though is that you can only see enemies by using a sonar wave ping. You must stealthily sneak around the security and collect all the key cards need to open the door and escape the mansion. Silver Echo was made by 8 person team in Unity 5 over 48 hours during Global Game Jam.

What I Worked On

  • Programmed the Pinging mechanic
  • Setup the Game Camera
  • Setup animations on Enemies
  • Worked on Enemy AI with Charles
  • Worked with Art team to import models
