Above and Beyond

For my Above and Beyond, I put links on all of my game pages to download executable copies of the games I made, as well as links to the GitHub for each of those projects. I also made a dropdown navigation bar and wrote some JavaScript to make it work when it becomes vertical. I also used GitHub track and version manage all the work I did on the project and included a link to it in the site if anyone wants to see the code for this website. I also included multiple screenshots of each game project so employers can see what the project looks like.
