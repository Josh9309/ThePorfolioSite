GitHub Tutorial Website

Summary

GitHub Tutorial Website is made to help teach new people how to use GitHub. I work as a Lab Worker for my college and I found it shocking how many people did not know what GitHub was or how to use it. I then decide to create this tutorial to help people who have no understanding of how GitHub works or what it is. I coded the whole site from scratch using HTML and CSS.
